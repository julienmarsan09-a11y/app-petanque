Guide Gmail

  1. Connectez-vous à votre compte Google
  2. Allez dans Sécurité → Validation en 2 étapes (activez-la si ce n'est pas fait)
  3. Dans Sécurité, cherchez Mots de passe des applications
  4. Créez un mot de passe pour "Pétanque" et copiez les 16 caractères générés
  5. Collez ce mot de passe dans le champ ci-dessus
Paramètres Gmail :
Serveur : smtp.gmail.com · Port : 587
